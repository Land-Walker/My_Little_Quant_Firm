# My Little Quant Firm

This is a project built upon three sub-projects:
1. FinD_Generator
2. R2_T2
3. HFT Simulator

As a whole, this project operates as a virtual quant firm by:
- FinD_Generator generates financial data to train trading bots
  - reflecting excessive data of quant firms
- R2_T2 are trading bots based on reinforcement learning who actually trade in the market
  - reflecting quant traders
- HFT simulator provides overall infrastructure of quant firms in simpler version to connect the market and orders made by trading bots
  - reflecting actual market and HFT infrastructure

Therefore, this project is basically to draw some vague pictures on my head about how quant firms work.
